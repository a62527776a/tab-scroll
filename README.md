# 关于TabScroll
TabScroll是一个基于Vue(2.X)和Better-Scroll的滚动库，主要满足web端的复杂滚动需求

由于依赖Better-Scroll，所以需要对Better-Scroll具有一定的了解。 如果还没有了解过，建议先看看Better-Scroll官方文档。
